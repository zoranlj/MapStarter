# MapStarter
## An Ionic 4, Angular 6, StencilJS MapStarter project powered by LeafletJS and Esri

### Run the project
Install the dependencies:
```bash
$ sudo npm install -g ionic cordova ios-deploy
```

Run the project:
```bash
$ ionic lab
```
